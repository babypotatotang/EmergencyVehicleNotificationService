# Project: Dajung-Web
* 긴급 자동차 위치 알림 서비스
* 2020 이브와 ICT멘토링
* 2020.05.13 ~ 2020.11.30

이 프로젝트는 긴급자동차로 인한 사고를 예방하기 위해서 고안된 프로젝트이다.
민원으로 인해 긴급자동차의 사이렌 소리가 줄어들고, 주변 소음이나 차 내부 소음으로 인해 도로 교통상황을 신속하게 파악하기 어렵다.
이로 인해 발생하는 사고들을 예방하기 위해 운전자 앱에서 긴급자동차의 위치를 파악할 수 있게 한다.
